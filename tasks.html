Практически задачи за GIMP

  • Задачи

Задача 1 — Създаване на постер с текст

  1. File → New (Файл → Нов), размер: 800x600 пиксела.
  2. Bucket Fill Tool (Инструмент за запълване) → запълнете фона.
  3. Text Tool (T) (Инструмент за текст) → напишете „Добре дошли в GIMP“.
  4. Tool Options (Настройки на инструмента) → шрифт, размер, цвят.
  5. File → Save As (Файл → Запази като) → poster.xcf.
  6. File → Export As (Файл → Експортиране като) → poster.png.

Задача 2 — Създаване на трицветно знаме

  1. File → New (Файл → Нов), размер: 1400x900 пиксела.
  2. Rectangle Select Tool (Инструмент за правоъгълна селекция) → три ленти.
  3. Bucket Fill Tool (Инструмент за запълване) → бяло, зелено, червено.
  4. File → Open as Layers (Файл → Отвори като слоеве) → герб.
  5. Move Tool (Инструмент за местене) → позиционирайте герба.
  6. File → Save As → Bulgaria.xcf, Export As → Bulgaria.png.

Задача 3 — Картичка със снимка и текст

  1. File → New (Файл → Нов), размер: 800x600 пиксела.
  2. File → Open as Layers (Файл → Отвори като слоеве) → снимка.
  3. Move Tool (Инструмент за местене) → позиционирайте снимката.
  4. Text Tool (T) (Инструмент за текст) → напишете „Привет“.
  5. File → Save As → card.xcf, Export As → card.png.

Задача 4 — Промяна на цветовете на снимка

  1. File → Open (Файл → Отвори) → снимка.
  2. Colors → Hue-Saturation (Цветове → Оттенък-наситеност).
  3. Colors → Brightness-Contrast (Цветове → Яркост-контраст).
  4. File → Save As → photo_edit.xcf, Export As → photo_edit.png.

Задача 5 — Изрязване на част от изображение

  1. File → Open → снимка.
  2. Rectangle Select Tool → селекция на частта.
  3. Image → Crop to Selection (Изображение → Изрежи до селекция).
  4. File → Save As → crop_example.xcf, Export As → crop_example.png.

Задача 6 — Рисуване с четка

  1. File → New → 600x400 пиксела.
  2. Paintbrush Tool (Четка) → изберете цвят и размер.
  3. Нарисувайте върху фона.
  4. File → Save As → paint_example.xcf, Export As → paint_example.png.

Задача 7 — Изтриване с гумичка

  1. File → Open → изображение.
  2. Eraser Tool (Гумичка) → изтрийте части от изображението.
  3. File → Save As → erase_example.xcf, Export As → erase_example.png.

Задача 8 — Преместване на слой

  1. File → Open → изображение с няколко слоя.
  2. Move Tool → изберете слой и преместете.
  3. File → Save As → move_example.xcf, Export As → move_example.png.

Задача 9 — Добавяне на ефект Blur

  1. File → Open → изображение.
  2. Filters → Blur → Gaussian Blur (Филтри → Размазване → Гаусов Blur).
  3. File → Save As → blur_example.xcf, Export As → blur_example.png.

Задача 10 — Комбинирана задача

  1. File → New → 800x600 пиксела.
  2. Rectangle Select Tool → направете селекция и запълнете с цвят.
  3. Text Tool → добавете надпис.
  4. Move Tool → преместете обекти по желание.
  5. Colors → Hue-Saturation → променете цвета.
  6. File → Save As → final_example.xcf, Export As → final_example.png.
